# Download most recent LendingClub data:
1. download the most recent Payments History Files (All Payments version) at
http://additionalstatistics.lendingclub.com/ 
2. Download the Loan Data from
https://www.lendingclub.com/info/download-data.action -- Make sure you're
logged in to your LendingClub account wwhen you do this; you get a more
restricted dataset if you are not logged in).  -- Download all of the files,
not just the one with the most recent loans, since these files contain current
loan status for all loans.  
3. Unzip all the files and save to data/loanstats/

In constants.py, in the class variables for the PathManager class, update the
"payments_file" variable to point to the current payments csv file.

"python loanstats.py" updates all the loan data caches 
"python curves.py" updates the default and prepayment curves
"python build_all.py" builds all the random forest models.

To push to prod, just copy entire training_data directory into a new directory,
and make sure the model_dir path in run_lc.py::main() points there.
